\DEBUG

Package: Default
A class for conditionally printing output to the screen. The debug class allows you to leave important debug statements in the code while centrally deciding whether they will print their output or not.
Output can also be buffered and then printed when desired. All of the functions in Debug work only if the class has been activated by a call to DEBUG::activate();
